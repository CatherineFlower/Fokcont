# Фокконт — мягкое снижение цифровых отвлечений при работе

**Фокконт** — это мобильное приложение для Android на языке Kotlin, ориентированное на повышение продуктивности и формирование осознанности в момент цифрового отвлечения.

## 📝 Описание проекта
В основе работы приложения лежит использование системного сервиса `UsageStatsManager`, который фиксирует факты запуска приложений. Вместо жестких блокировок, Фокконт предлагает **«прерывание автоматического поведенческого паттерна»**. При попытке открыть отвлекающее приложение система выводит интерфейс с предложением сделать паузу, выполнить упражнение или вернуться к задаче.

## 🚀 Основной функционал

### 1. Фокус-сессии и трекинг
Запуск сессии позволяет приложению начать мониторинг активности в фоновом режиме.
- **Локальное отслеживание**: фиксация переключений на сторонние приложения.
- **Таймер**: визуализация времени текущей фокусировки.
- **Заметки**: возможность добавить описание к каждой сессии.

> **Screenshot 1: Главный экран сессии**
> *Подпись: На этом экране пользователь запускает таймер и видит текущее количество переключений.*

### 2. Умное прерывание (Interruption UI)
Если пользователь открывает отвлекающее приложение из своего списка:
- Появляется экран осознанности.
- Предлагаются варианты: «Вернуться к работе», «Все равно открыть».



### 3. Гибкая настройка контроля
Пользователь сам решает, какие приложения считать отвлекающими.
- Полный список установленных приложений.
- Сохранение настроек в локальное хранилище.

> **Screenshot 3: Выбор приложений для контроля**
> *Подпись: Интерфейс выбора приложений, которые система будет отслеживать.*

### 4. Аналитика и Статистика
Визуализация продуктивности за неделю.
- График переключений по дням.
- Общая статистика: средняя длительность сессии, общее время фокуса.

> **Screenshot 4: Экран статистики**
> *Подпись: Гистограмма активности, построенная на реальных данных из базы Room.*

### 5. Конфиденциальность и Безопасность
- **Локальное хранилище Room**: данные не передаются на внешние серверы.
- **Шифрование AES-256**: защита локальных данных.
- **Удаление аккаунта**: полная очистка всех данных с устройства в один клик.

> **Screenshot 5: Настройки и удаление данных**
> *Подпись: Раздел настроек с управлением доступом и функцией полного удаления данных.*

## 🛠 Технологии
- **Язык**: Kotlin
- **БД**: Room (SQLite)
- **Архитектура**: MVVM (ViewModel, Repository)
- **Навигация**: Jetpack Navigation Component
- **Сервисы**: UsageStatsManager, Foreground Service (Android 14+ compatible)
- **UI**: Material Design 3, View Binding

## 📂 Установка
1. Клонируйте репозиторий.
2. Откройте в Android Studio.
3. Соберите проект (`./gradlew assembleDebug`).
4. **Важно**: После запуска предоставьте разрешение "Доступ к статистике использования" (Usage Access) в настройках.
